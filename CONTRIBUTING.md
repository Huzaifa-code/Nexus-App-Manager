# Contributing to Nexus App Manager

First off, thank you for considering contributing to Nexus App Manager! It's people like you that make Nexus App Manager such a great tool.

## How Can I Contribute?

### Reporting Bugs
* **Check the Issues**: Before reporting a bug, check if it's already been reported.
* **Be Specific**: Include details about your environment (Linux distribution, Desktop Environment, etc.).
* **Provide Logs**: If the app crashes, try to provide the terminal output.

### Suggesting Enhancements
* **Explain the Value**: Why would this feature be useful to others?
* **Be Descriptive**: Detail how it should work.

### Pull Requests
1. **Fork the repo** and create your branch from `main`.
2. **Install dependencies** using `npm install`.
3. **Ensure your code builds** with `npm run tauri build`.
4. **Follow the code style** (Prettier/ESLint are recommended).
5. **Issue a Pull Request** with a clear description of your changes.
